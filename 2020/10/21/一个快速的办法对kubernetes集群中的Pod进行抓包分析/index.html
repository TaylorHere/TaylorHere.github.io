一个快速的办法对kubernetes集群中的Pod进行抓包分析

为你的kubectl安装plugin manager

krew

完成安装后运行命令

kubectl krew

我喜欢将shell命名alias为简短的缩写

alias kk="kubectl krew"

下载安装ksniff插件

kk install sniff

下载安装WireShark

使用sniff抓包

kk sniff <pod> -n <namespace>

他会自动打开wire shark界面

一件抓包调试不是梦
